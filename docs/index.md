Welcome to the official documentation for **Prompter AI v2.0**, your intelligent prompt engineering co-pilot embedded directly inside supported AI websites.

## 🌟 Feature Highlights

### 1. Smart Prompt Interview Mode
Never write bad prompts again. Prompter AI v2.0 evaluates your prompt completeness score. If it falls below 75, the assistant slide-in displays 1-3 targeted questions (art styles, language constraints, target audience) to inject optimal context automatically.

### 2. Draggable floating widget
The interactive widget rests on the bottom-right corner of ChatGPT/Gemini pages, and is fully draggable with automatic position memory. It remains active across all Single Page Application (SPA) navigation changes.

### 3. Multi-Provider BYOK Integration
Route requests through Gemini, OpenAI, Claude, Groq, or OpenRouter. Your keys stay on your machine.

### 4. Interactive Side Panel
Analyze intent, read word-diff changes side-by-side, save favorites, view usage analytics, and templates.
